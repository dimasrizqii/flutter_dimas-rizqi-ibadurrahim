# **(15) Flutter Layout**

Terdapat beberapa macam materi yang harus kita ketahui tentang dasar dari bahasa Pemrograman Dart ini, antara lain :
>
> 1. Apa itu Layout?
> 2. Single-child Layout
> 3. Multi-child Layout

## **Apa itu Layout?**

- Deskripsi

> Layout digunakan untuk mengatur tata letak. Layout sendiri berbentuk Widget yang dapat mengatur Widget yang berada di dalamnya.

## **Single-child Layout**

- Container

> Container digunakan untuk membuat sebuah box, container juga dapat membungkus widget lain, container bisa dianggap sebagai sebuah box yang memiliki margin, padding, dan border.

- Center

> Center digunakan untuk membuat sebuah box, center juga dapat membungkus widget lain, center akan memenuhi lebar dan tinggi ruang di luar center ini sendiri.

- SizedBox

> SizedBox digunakan untuk membuat sebuah box, container juga dapat membungkus widget lain, box yang berada di SizedBox dapat diatur lebar dan tingginya. Secara keseluruhan, SizedBox lebih sederhana dibandingkan container.

## **Multi-child Layout**

- Column

> Column digunakan untuk mengatur widget secara vertikal.

- Row

> Row digunakan untuk mengatur widget secara horizontal.

- ListView

> ListView dapat mengatur widget dalam bentuk list. Jika kita menggunakan ListView, maka otomatis data kita memiliki kemampuan untuk melakukan scroll.

- GridView

> GridView digunakan untuk mengatur widget dalam bentuk kategori.
